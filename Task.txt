Вимоги до тестового завдання:
	- HTML5
	- CSS
	- Flex-box
	- Responsive
	- Можна використовувати будь-які фреймворки типу Bootstrap, Foundation  
	- Для слайдерів підключити будь-яку бібліотеку, або написати власний слайдер (за бажанням)
	- Картинки для слайдерів можна вибрати будь-які
	- Плюсом буде використання CSS препроцесорів (SASS, SCSS, LESS)

Даний макет завантажений з одного з інтернет ресурсів, що надає безкоштовні макети для сайтів, та не являється комерційною одиницею.
